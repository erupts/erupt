# erupt-magic-api

Dynamic HTTP API builder for Erupt, powered by [magic-api](https://gitee.com/ssssssss-team/magic-api).

Create, edit, and publish REST endpoints at runtime through a web IDE — no recompilation or restart needed. Scripts execute against the application's datasources. Integrates with `erupt-upms` for access control on both the editor UI and the generated endpoints.
