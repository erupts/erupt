# erupt-ai

LLM and MCP integration for Erupt, powered by [LangChain4j](https://github.com/langchain4j/langchain4j).

Supports multiple providers out of the box: OpenAI, Anthropic Claude, Google Gemini, and Ollama (local models). Also implements the **Model Context Protocol (MCP)** for tool-calling and agent workflows. Exposes AI capabilities as Erupt-managed admin UI entries — no extra controller code needed.
